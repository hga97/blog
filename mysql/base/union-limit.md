### limit的使用

mySql 提供了 limit ，主要用于提取前几条或者中间某几行数据

?> select * from table limit m,n  
其中 m 是指记录开始的 index，从 0 开始，表示第一条记录  
n 是指从第 m+1 条开始，取 n 条。

#### 取得前5条数据

```bash
select * from emp limit 5;
```
